Sometime last week, I had a wild hair to look at Python again. So that's pretty much all I've been doing over the last week; trying to absorb as much info as possible to get a better understanding of it. In particular, object oriented programming topics. That seems to be an ongoing area of confusion for me. What do I plan on doing with Python, though? For now, scripting and simple apps/web apps. No real plans as of yet, but I can definitely see how it will be useful in the long run. 

Outside of that, Javascript is also on my plate. It seems no matter where I go, it comes up. Front end, back end, server side, and now even music programs. For example, I was looking at [Bitwig](http://www.bitwig.com/en/bitwig-studio.html) last night, and noticed that they have an open controller API that's all javascript based. You can use it to control parameters and gear. Also, [Reaper](http://www.reaper.fm/sdk/js/js.php) has a JSFX, or javascript based system for effects where you can code your own sample and instrument effects. Definitely worth checking out, too. 

For this week, I picked up a copy of the latest Somatic Responses album, Folding Space. It's amazing that they've put consistently put out music for almost 2 decades now, and each release still maintains the same level of quality. This album is definitely right on par with the most recent releases, and most folks seem to compare it to Concrete Glider. Dark ambient strings, 909s, idm structures. Check it out:
