Preacher season 2 is back on, and I'm pretty stoked. Last season was surprisingly very good despite what some of the critics were saying; but then again, when do I ever listen to what critics say? Most of the movies that I like are rated around 6 stars on [IMDB](https://www.imdb.com)...not that that tells you a lot about the show--just saying.

Anyways, I'm still also on my goal to land some sort of ops/devops type job so I've been hustling on some online learning programs. Most of the command line type stuff I feel pretty comfortable with, it's just the monitoring tools and programs that I need to get my feet wet. Onward and upward, right?

Software wise, I've made some tweaks to my setup. Being a long time MATE desktop user, I recently switched to Gnome after installing the new [Pop!_OS](https://system76.com/pop) distro by the folks at System76. I'm really loving the flat, modern, theme they have going so I installed Gnome along with that theme separately. You can find that [here](https://github.com/system76/pop-gtk-theme) if you want to check it out without installing a full distro.

Also, I've been seeing screenshots occasionally of other folks terminal setups and noticed that some folks had some interesting looking colorized command lines that outlined git branches and statuses. So I did some digging and came across various Bash themes, specifically [agnoster.bash](https://github.com/speedenator/agnoster-bash). It's pretty straightforward to setup on Linux, but took some trial and error to get working on Mac. I'm pretty happy with the results though.

Not a whole lot going on music wise that I'd like to share, but there's a new Huron album to be released soon. He's released a couple of tracks from the album that can be found below.
