Learning, learning, learning. That's what I spend a lot of my free time on; anything and everything that strikes me. The thing is that I get a lot more free time over the weekend to explore ideas, and dig into new things. Then Monday morning rolls around and it's back to work as usual. All the drama and mental exhaustion that goes into my work drains me through the week. It's an endless cycle. While I haven't mastered anything, I've amassed a good, wide, range of topics. Which has its advantages, so the next project or topic that I start on, I feel like I have a head start. It's good and it's bad, really. 

Now that that's off my chest, I also opt-out of learning Flask since the last post so that I could go back to Django. Both frameworks are really nice. Flask is a no-fuss, no-muss, framework. Light, simple but requires plugins and things to get you going. Django has so much more going for it right out of the box that I think I'm leaning more toward working with it for awhile. 

I also took a couple of days to create a simple script that will automatically generate a folder, create an index.html and main.css file, along with an assets folder for css, js, and img. It then downloads jquery and normalize.css. It's not anything to boast but it gets the job done. What I would like to do is clean up the code a bit so that it's not so procedural. Object oriented paradigms are the way to go, right? Then I want it to download the current versions of normalize.css and jquery. Right now, it's just a set, static, version. Still much to learn and do. Feel free to give that script a whirl [here](https://github.com/terribledevice/python_scripts/blob/master/webs.py) if it tickles you. 

Linux has also taken over my life. There was a job posting for a Jr. Systems Engineer that I saw recently. After looking over all of the skill sets that they're looking for, I really think that it would be a good fit for me. I'd say that I'm pretty confident in my abilities to work at the command line. Log files, monitoring, and system processes I'm Ok with--though something I'd want to get a little more experience with. This will probably be on my mind for awhile so maybe I'll give it some more thought.
