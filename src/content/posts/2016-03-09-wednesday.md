Earlier in the week, I finally finished a Python script to help me out at work. The script is written in Python3 and will generate a list of email aliases for a user-defined amount. The use case is that, say you need to test out a signup form, or some email list where you can receive multiple emails. Well, since an email alias is treated as a new email, but messages will be received at the original address. This script can generate multiple copies of it. Check out my Python scripts here, and feel free to fork or give a Pull request:[Python_scripts](https://github.com/terribledevice/python_scripts)
